# glass-apps

Web apps for **Meta Ray-Ban Display** glasses (600×600 in-lens display + Neural Band input).

| App | What it is | Play |
|---|---|---|
| [ShadowJack](./shadowjack/) | Discreet blackjack — "the game nobody sees." Hit, stand, and double with tiny Neural Band finger gestures while your hand rests at your side. 6-deck shoe, 3:2 blackjack, persistent bankroll. | `/shadowjack/` |

## Add an app to your glasses

1. The app must be live at a public HTTPS URL (these are, via GitHub Pages).
2. On the phone paired with your glasses, open the **Meta AI** app → **Devices** → your Display glasses → **App connections** → **Web apps** → **Add a web app** → enter a name and the app's URL.
3. The app appears in the glasses' app library. Re-deploys to the same URL update automatically.

Controls in all apps: Neural Band swipes = arrow keys, index-finger pinch = select, middle-finger pinch = back.
